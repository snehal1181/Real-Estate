================================================================================

    Contact page for visual consistency with all other CTA buttons.

[1.3.0] - 2026-04-02
================================================================================
ADDED
  - Premium "Silk & Gold" animated canvas background implemented globally
    across all pages.
  - Frosted-glass effect applied to all content sections to ensure readability
    over the animated background.

FIXED
  - Resolved issue where the animation overlay incorrectly blocked user
    interaction with page content.

================================================================================
[1.2.0] - 2026-04-01
================================================================================
ADDED
  - Luxury "Diamond Glimmer" canvas-based particle animation engine.
    - Twinkling diamond and gold dust particles react to mouse movement.
    - High-performance, RAF-based render loop for 60fps animation.
  - Global page loading indicator (preloader) with animated SVG home icon
    ("Loading Excellence" text):
    - Waving concentric circle rings.
    - Outlined house/home path icon in the centre.
  - Dark / Light theme toggle button (moon/sun icon) in the navbar.
    - User preference persisted in `localStorage`.

REPLACED
  - Basic CSS particle animation replaced with the new Canvas-based Diamond
    Glimmer effect for a significantly more premium aesthetic.

================================================================================
[1.1.0] - 2026-03-31
================================================================================
ADDED
  - Properties page (properties.html):
    - Full property catalogue with 12+ listings.
    - Client-side filter bar (location, type, price range).
    - URL-parameter-based search routing from the Home page search box.
    - Pagination control (6 properties per page) with prev/next navigation.
    - "No results found" empty state with clear-filter action.
    - "Clear Filters" button displayed only when active filters are applied.
  - Property Detail Modal (available on both Home and Properties pages):
    - Full-screen Bootstrap modal with image carousel (exterior + interiors).
    - Displays: title, price, location, beds, baths, sqft, and description.
    - Images dynamically injected via JavaScript data attributes.
  - Cart / Add-to-Cart page with professional luxury layout:
    - Two-column grid: cart items list + order summary sidebar.
    - Trust badges (secure payment, free consultation, verified listings).
    - Quantity adjustment and item removal controls.

IMPROVED
  - Standardised all property card image dimensions for visual consistency.
  - Enhanced overall cart page design to match the premium brand identity.

    - Dark/light theme toggle with localStorage persistence.
    - IntersectionObserver scroll animations.
    - Search form → properties.html URL routing.
    - Properties page filtering and pagination logic.
    - Property detail modal data injection and carousel population.
    - Mobile navbar auto-collapse on link click.
  - css/ — Page-specific stylesheets:
    - home.css, about.css, properties.css, contact.css
